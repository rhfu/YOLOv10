# YOLOv10 训练自己数据集

基于YOLOv10训练自己数据集


## 安装

- git clone git@github.com:THU-MIG/yolov10.git
- cd yolov10/

  
## 使用说明

- 复制 tran.py、test.py、mydata到yolov10根目录
- tran.py 为训练文件
- test.py 测试文件
