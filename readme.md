# YOLOv10 训练自己数据集

基于YOLOv10训练自己数据集

## 安装

- git clone git@github.com:THU-MIG/yolov10.git
- cd yolov10/

## 使用说明

- 复制 train.py、test.py、mydata到yolov10根目录
- python train.py # 训练
- python test.py # 测试
